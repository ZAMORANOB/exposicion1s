Cuestionario de HTML y Diseño Web

1. ¿A qué se denomina Multimedia, hipermedia e hipertexto?

  • Multimedia: Combinación de texto, imágenes, audio y video.
  • Hipermedia: Multimedia con enlaces que permiten navegar.
  • Hipertexto: Texto enlazado para navegar entre contenidos.

2. ¿Qué es una Página Web? ¿Qué editores de texto se usan?

  • Es un documento digital que se muestra en un navegador web.
  • Se usan editores como Notepad++, Visual Studio Code, Atom y Sublime Text.

3. ¿Qué significa HTML y para qué se lo utiliza?

  • Significa HyperText Markup Language.
  • Se usa para estructurar el contenido de páginas web.

4. ¿Qué etiquetas marcan el inicio y final de un documento HTML?

  • <html> y </html>
  • <!DOCTYPE html> define que es HTML5.

5. ¿Las etiquetas <BODY> y </BODY>, qué indican dentro del documento HTML?

  • Delimitan el cuerpo visible del documento HTML.

6. ¿Cuáles etiquetas se han declarado dentro de <HEAD> y </HEAD>?

  • <title>, <meta>, <link>, <style>, <script>

7. ¿Cuáles son las etiquetas de los atributos de texto? Ejemplo de cada uno:

  • <b>Negrita</b>
  • <i>Cursiva</i>
  • <u>Subrayado</u>
  • <strong>Importante</strong>
  • <em>Énfasis</em>
  • <small>Texto pequeño</small>

8. ¿Cuál es la etiqueta que participa como parámetro en otras etiquetas y qué función cumple?

  • Los atributos como style, class o href funcionan como parámetros.
  • Sirven para modificar el comportamiento o estilo de una etiqueta.

9. ¿Cuáles son las formas de utilizar las etiquetas de listas? Ejemplos:

  • <ol> – Lista ordenada:
    • <ol>
  <li>Paso 1</li>
  <li>Paso 2</li>
</ol>
  • <ul> – Lista desordenada:
    • <ul>
  <li>Pan</li>
  <li>Leche</li>
</ul>
  • <dl> – Lista de definición:
    • <dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado</dd>
</dl>

10. ¿Cuál sería la forma de colocar un color de fondo a la página web?

  • Con atributo style en <body>:
    • <body style="background-color: lightblue;">

11. ¿Para qué se utiliza la etiqueta <FONT ...>? Aplicar en un ejemplo:

  • Se usaba para cambiar color y fuente, pero está obsoleta.
  • Ejemplo obsoleto: <font color="blue">Texto</font>
  • Forma moderna: <span style="color: blue;">Texto</span>

12. ¿Qué diferencia hay entre usar <P> y <DIV>?

  • <p> es para párrafos, tiene significado semántico.
  • <div> es un contenedor genérico sin significado semántico.

13. ¿Cuál es la estructura básica del código HTML?

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Título</title>
  </head>
  <body>
    <h1>Encabezado</h1>
    <p>Contenido</p>
  </body>
</html>

14. ¿Cuáles son las etiquetas de encabezamiento o título?

  • <h1> a <h6> – De mayor a menor tamaño e importancia semántica.
